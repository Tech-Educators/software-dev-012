flex-direction

Default: row

This determines what the main axis is.

When set to "row", the main axis is X and the cross axis is Y

When set to "column" the main axis is Y and the cross axis is X.

Our flex items (the children) will be positioned along the main axis.

justify-content

Default: flex-start

This aligns the flex items (children) along the main axis

align-items

Default: flex-start

This aligns the flex items (children) along the cross axis
